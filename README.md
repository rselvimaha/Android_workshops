ImageDownload Project:
  This project demonstrates how to download an image from the internet using a background thread and display it in an Android application.
It reinforces key Android concepts such as networking on a background thread, UI updates on the main thread, and permission handling.

RecordAudio Project:
  This project demonstrates how to build an Android application that records audio using the device microphone after obtaining the required permissions, and then plays back the recorded audio. The exercise focuses on: Runtime permission handling Using MediaRecorder to record audio Using MediaPlayer to play back audio Emulator microphone configuration
Application Features: 
When the application is launched:
The user interface displays: Start Recording button Stop Recording button Clicking Start Recording: Requests microphone permission (if not already granted) Starts audio recording after permission approval Clicking Stop Recording: Stops the audio recording Plays back the recorded audio
